> 技术栈：微信原生小程序 + 微信云开发（CloudBase）

## 管理后台

管理后台内嵌于小程序中，通过手机号白名单鉴权，学员端无感知。

### 进入方式
「我的」页面 → **长按昵称 5 次** → 进入管理员登录页

### 管理员开通
1. 在云开发控制台 → 数据库 → 创建 `admins` 集合
2. 添加记录：`{ "phone": "手机号", "name": "姓名", "role": "super" }`
3. 管理员用该手机号在小程序登录即可

### 功能
- 题库管理：增删改查题目，支持单选/判断/多选三种题型
- 学员管理：查看学员答题次数、正确率、最近活跃
- 数据统计：题量分布、题型分布、平均正确率

## 快速开始

### 1. 导入项目
1. 打开**微信开发者工具**
2. 选择「导入项目」
3. 项目目录选择 `xinyue-quiz-miniprogram` 文件夹
4. AppID 填入你的小程序 AppID（或选择「测试号」体验）

### 2. 开通云开发
1. 在开发者工具点击「云开发」按钮，开通环境（免费）
2. 复制云环境 ID
3. 打开 `miniprogram/app.js`，把 `your-cloud-env-id` 替换为你的环境 ID

### 3. 创建数据库集合
在云开发控制台 → 数据库，创建以下集合（详见 `database.md`）：
- `questions` — 题目库
- `records` — 答题记录
- `wrong_questions` — 错题本

### 4. 部署云函数
右键 `cloudfunctions/` 下的每个文件夹 → 「上传并部署：云端安装依赖」

### 5. 体验
导入后即可看到首页（显示叉车司机科目，1000题），点击进入答题。**无需配置云开发即可离线答题**，题目已内置在 `data/questions.js`。
